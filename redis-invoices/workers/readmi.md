**Добавление новых МП** 

1) Скопировать готовый файл одних из МП
2) Зайти поменять доступы (ID МП и Путь до генерации счетов)

ID МП - это id который указан у группы этого МП 

"Путь до генерации счетов" - файл который находится в папке mp_handlers 


**Создание файла генерации:**

1) Скопировать один из готовых файлов и переименовать. Можно указать ID мп в названии
2) Логику вычисления нужно писать внутри
